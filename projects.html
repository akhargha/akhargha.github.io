My Projects

  1. Course Scheduler and Sharing Tool:
    2. HTML, CSS, Python (BeautifulSoup & pandas), SQL

    This is my long-term personal project with which I aim to develop a useful product for college students while learning full stack development. This is a course scheduler and sharing tool to streamline the process of course scheduling and planning events/meetings with friends. This scheduler adds visualizations to a schedule and acts as a platform to share schedules with friends so that they are aware of time constraints when making plans (and vice-versa!)

    Current Progress:
    • Developed a web-scraper using BeautifulSoup and pandas to scrape course data from college's website.
    • Designed a 5-entity relational database for the scheduler using SQL.
    Future Work:
    • Implement user authentication.
    • Implement API endpoints for user registration, login, CRUD operations for courses, and friend functionalities.


  2. Usable PKI Browser Extension:
    3. HTML, CSS, JavaScript

    This is a browser extension I am developing as a part of my research project on Cybersecurity. The goal of the extension is to make the Public Key Infrastructure usable for everyone and to prevent phishing attacks. The extension uses browsing history and web-certificates to detect phishing attacks. This project has taught me a lot about building large-scale applications while self-learning languages and frameworks.

    Future Work:
    • Backend development- implement API endpoints to collect user data. Modify frontend to facilitate that.
    • Improve the UI and UX of the extension.
